3+|[#key-prefix-toc-dot-leader]*Key Prefix:* <<key-prefix-toc-dot-leader,toc_dot_leader>>

|content
|<<quoted-string,Quoted string>> +
(default: '. ')
|toc:
  dot_leader:
    content: ". "

|font_color^[2]^
|<<colors,Color>> +
(default: _inherit_)
|toc:
  dot_leader:
    font_color: #999999

|font_style^[2]^
|<<font-styles,Font style>> +
(default: normal)
|toc:
  dot_leader:
    font_style: bold

|levels^[3]^
|all {vbar} none {vbar} Integers (space-separated) +
(default: all)
|toc:
  dot_leader:
    levels: 2 3

|===

. `<n>` is a number ranging from 1 to 6, representing each of the six heading
levels.
. The dot leader inherits all font properties except `font_style` from the
root `toc` category.
. 0-based levels (e.g., part = 0, chapter = 1). Dot leaders are only shown
for the specified levels. If value is not specified, dot leaders are shown
for all levels.
