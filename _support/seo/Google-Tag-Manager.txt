

== Google Tag Manager installieren

Konto:   JekyllOne
GTM-ID:  GTM-KMT3P8N

Kopieren Sie den unten stehenden Code und fügen Sie ihn auf jeder Seite
Ihrer Website ein.

Fügen Sie diesen Code so weit oben wie möglich im <head> der Seite ein.

<!-- Google Tag Manager -->
<script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
})(window,document,'script','dataLayer','GTM-KMT3P8N');</script>
<!-- End Google Tag Manager -->

Fügen Sie diesen Code außerdem direkt nach dem eröffnenden <body>-Tag ein:

<!-- Google Tag Manager (noscript) -->
<noscript><iframe src="https://www.googletagmanager.com/ns.html?id=GTM-KMT3P8N"
height="0" width="0" style="display:none;visibility:hidden"></iframe></noscript>
<!-- End Google Tag Manager (noscript) -->
