<html>

Add the following TXT record to the https://jekyll.one domain DNS (Netlify):

Make sure that the URL of the site for which you are verifying your rights
in Yandex Webmaster matches the URL of the site for which you are creating
a DNS record. For example, URLs with and without the "www" prefix are
considered different sites. Updating records can take a long time.
If the site is running over IPv4 and IPv6, make sure that the site responds
correctly using all IP addresses. When the DNS records are updated,
click Check.
