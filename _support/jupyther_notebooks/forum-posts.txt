Hi Guys,

I'm Juergen, a web developer. Currently, I'm working on integrating Jupyter Notebooks (Python) in webpages to make Notebooks live/interactive available using the public Binder service for free.

I've integrated the (Python/JS) package/library nbinteract with my Jekyll-based
static website at https://jekyll.one. You'll find in the menubar a dropdown Live Examples. Go for one of these sites. On these sites, use the menu  Learn/Experimental/Textbooks and open one of the pages in that dropdown. The pages integrate some nice Jupyter Notebooks using interactive widgets.

You can create a personal website in notime if you're already on Github or Netlify. Go for the Rocketstart on https://jekyll.one to create your web!
